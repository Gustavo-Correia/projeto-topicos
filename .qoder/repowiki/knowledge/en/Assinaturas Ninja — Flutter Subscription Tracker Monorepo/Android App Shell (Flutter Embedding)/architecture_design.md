Single-activity Android app built with the Flutter Gradle plugin (`dev.flutter.flutter-gradle-plugin`) in `app/build.gradle.kts`; `MainActivity` extends `io.flutter.embedding.android.FlutterActivity`, so all UI logic lives in the Dart/Flutter layer above. The manifest registers `.MainActivity` as the exported LAUNCHER activity with `singleTop` launch mode and forwards to Flutter's NormalTheme via a meta-data entry; `flutterEmbedding=2` enables the v2 embedding. Build targets Java/Kotlin 17, pulls compileSdk/ndkVersion/minSdk/versionCode/versionName from the Flutter tooling via `flutter.*` properties, and points `flutter { source = "../.." }` at the repository root where the shared Dart code resides. Resources under `src/main/res/` provide only the default launch background and mipmap icons — no custom Kotlin/Java business code is present beyond the generated `GeneratedPluginRegistrant.java`.
