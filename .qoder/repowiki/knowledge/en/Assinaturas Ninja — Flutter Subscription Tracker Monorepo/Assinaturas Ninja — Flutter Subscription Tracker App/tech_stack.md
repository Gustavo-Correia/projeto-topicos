Flutter Material 3 (`useMaterial3: true`), state via `provider` (`ChangeNotifierProvider`/`Consumer`), persistence via `hive_flutter` with manual `jsonEncode`/`jsonDecode` over `Box<String>` keys, UUID generation via `uuid`, locale set to `pt_BR` through `intl`, HTTP via the `http` package for REST consumption (AwesomeAPI USD→BRL exchange rate and Banco Central SGS annual Selic rate), and named-route navigation via a central `onGenerateRoute`.
