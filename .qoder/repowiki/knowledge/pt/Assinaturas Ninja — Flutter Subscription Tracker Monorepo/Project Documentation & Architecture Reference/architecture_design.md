Coleção plana de documentos Markdown descrevendo a árvore fonte `lib/` irmã; nenhum código executável. `ARCHITECTURE.md` é o mergulho profundo canônico (inventário de arquivos, decomposição camada por camada com diagramas Mermaid, grafo de dependências, resumo de cobertura de testes). `README.md` fornece uma visão geral de início rápido e comandos de execução. `UI_GUIDE.md`, `IOS_GUIDE.md`, `VALIDATION.md`, `STRENGTHS_IMPROVEMENTS.md` e `TASKS.md` são suplementos específicos por tópico. `reference/CLAUDE_PROMPT.md` é um prompt de bootstrap para agente de IA que enumera quais docs ler primeiro e lista prioridades/restrições do projeto — é intencionalmente separado do README principal para não poluir a página inicial do repositório.
